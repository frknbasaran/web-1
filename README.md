# sausozluk / web

> web client implementation for sausozluk which is developed with Backbone.

## requirements

```sh
$ sudo npm install -g grunt-cli bower
```

## install

```sh
$ npm install
$ bower install
```

## config

> app/config.js

## run (build)

```sh
$ npm start
```

## run (test)

```sh
$ npm test
```
